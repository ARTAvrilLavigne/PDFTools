# PDFTools<br>
实现任意PDF文件分割和合并功能的windows图形化界面小工具<br>  

# 提供两个版本<br>  
1、若windows已安装java环境，双击运行`PDFSwing.jar`即可使用(`./PDF运行工具目录/PDFSwing.jar`)<br>  

2、(小白推荐版本)若windows未安装java环境，则使用`绿色运行版.rar`压缩包解压后双击执行.exe即可直接使用，集成了jre环境在内(`./PDF运行工具目录/绿色运行版.rar`)<br>
